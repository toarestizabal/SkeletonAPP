# MetaFit

App de seguimiento de ejercicios y alimentación.

## Características
- Registro de rutinas de ejercicios
- Control de ingesta calórica
- Progreso semanal
- Interfaz móvil responsive

## Tecnologías
- Ionic Angular
- TypeScript
- SCSS

## Instalación
```bash
npm install
ionic serve
